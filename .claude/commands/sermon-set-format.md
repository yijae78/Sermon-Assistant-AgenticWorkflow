# /sermon-set-format — HITL-5a: Manuscript Format

Set manuscript format and length.

## Checkpoint: HITL-5a

## Options
```
[Manuscript Format]
- Full Manuscript: Complete sentences throughout
- Semi-Manuscript: Key sentences + detailed notes
- Outline with Script: Structured outline + key quotes/transitions scripted

[Sermon Length]
- 15 min (~2,000-2,500 chars)
- 20 min (~2,800-3,200 chars)
- 30 min (~4,000-4,800 chars) [default]
- 40 min (~5,500-6,500 chars)
- Custom: ___ min

[Style Preference]
- Formal (worship atmosphere)
- Conversational (friendly atmosphere)
- Lecture (teaching atmosphere)
```

## After Selection
1. Record in session.json
2. Update state.yaml
3. Dispatch @sermon-writer for draft
